# QuickCV
Interactive Resume Builder -  A frontend-only web app built with HTML, CSS, and Vanilla JavaScript that lets users create resumes with live preview, dynamic sections, and PDF export support.
Interactive Resume Builder is a lightweight, frontend-only mini project built using HTML, CSS, and Vanilla JavaScript.
It allows users to create a professional resume by filling a simple form and instantly seeing a live preview of their resume.

The project supports dynamic sections like Education and Experience, a real-time progress bar to track form completion, and skill tags generated automatically from comma-separated input. Users can also export their resume as a PDF using the browserâ€™s print functionality.

This project demonstrates strong fundamentals of DOM manipulation, event handling, dynamic UI rendering, and responsive design, making it ideal for students and beginners building portfolio projects.

Key Features:

Live resume preview while typing

Add/remove multiple education and experience entries

Auto skill chips generation

Form completion progress bar

Print-friendly PDF download

Clean, responsive UI
